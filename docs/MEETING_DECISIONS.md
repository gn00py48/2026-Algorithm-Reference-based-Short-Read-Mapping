# Team Meeting Decisions

## Project Direction

- Problem: Reference-based Short Read Mapping
- We do not implement Denovo Assembly in the main project.
- We do not implement BWT/FM-index in the main project.
- Future work may mention Denovo, BWT/FM-index, paired-end reads, reverse complement, and DP alignment.

## Algorithms

1. Native Sliding Mapping
2. KMP-based Seed Search
3. Hash-based Seed-and-Extend
4. Weighted Hash Seed-and-Extend

## Shared Assumptions

- DNA alphabet: A, C, G, T
- Reference genome R is randomly generated.
- Personal genome G is generated by applying SNP mutations to R.
- Short reads are sampled from G.
- trueStart is used only for evaluation, never inside mapping algorithms.
- Only substitution mismatches are considered.
- Insertions and deletions are excluded.
- Reverse complement is excluded.

## Default Parameters

| Symbol | Meaning | Default |
|---|---|---:|
| N | reference length | 10,000 |
| M | read count | 1,000 |
| L | read length | 50 |
| D | allowed mismatch | 2 |
| SNP rate | R-to-G mutation rate | 0.3% |
| read error | per-read sequencing error | 0~1 |

## Seed Rule

- seedCount = D + 1
- seedLength = L / seedCount
- the last seed includes the remaining length

Example:

- L = 50
- D = 2
- seedCount = 3
- seed lengths = 16, 16, 18

## Candidate Start Rule

candidateStart = seedPos - seedOffset

Exclude invalid candidates:

- candidateStart < 0
- candidateStart + L > N
- duplicated candidateStart

## Evaluation Metrics

- Running Time
- Exact Position Accuracy
- Valid Mapping Accuracy
- Average Candidate Count
- Average Mismatch Count
- Index Size / Memory Usage
